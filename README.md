Jenkins Cookbook
================

This cookbook installs jenkins on your Ubuntu or Debian system

Usage
-----
e.g.
Just include `jenkins` in your node's `run_list`:

```json
{
  "name":"my_node",
  "run_list": [
    "recipe[jenkins]"
  ]
}
```
